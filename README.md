## Estructura

```
index.html      ← Sitio estático principal (GitHub Pages)
images/         ← Capturas de pantalla de la app
README.md       ← Este archivo
```

## Cómo actualizar

1. Editar `index.html` con las nuevas funcionalidades
2. Commit y push a `main`
3. GitHub Pages se actualiza automáticamente

## Configurar GitHub Pages

1. Ir a **Settings → Pages** en este repositorio
2. Source: **Deploy from a branch**
3. Branch: `main` / `/ (root)`
4. Guardar
